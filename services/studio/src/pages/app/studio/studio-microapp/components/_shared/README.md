# Shared Configurations

Common JSON configurations used across multiple components.

## Files:
- `typography.config.json` - Typography properties (font-size, font-family, etc.)
- `size.config.json` - Size properties (width, height, padding, margin)
- `border.config.json` - Border properties (border, border-radius)
- `common-inputs.config.json` - Common input properties (value, id)
